Abstract

The XModAlg package provides functions for computation with crossed modules of commutative algebras and cat\(^{1}\)-algebras). The current version is 1.16, released 29nd January 2018 for GAP 4.8.

Bug reports, suggestions and comments are, of course, very welcome. Please submit an issue on GitHub at http://github.com/gap-packages/xmodalg/issues/ or contact the second author at aodabas@ogu.edu.tr.
